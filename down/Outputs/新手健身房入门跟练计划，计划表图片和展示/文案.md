1️⃣ Day 1：胸 + 三头肌 👙
✅ 哑铃卧推：4组 x 12次（重量选择做完还能做2个的）
✅ 哑铃飞鸟：4组 x 12次（感受胸肌的拉伸）
✅ 绳索下压：4组 x 15次（拜拜肉克星 🔥）

2️⃣ Day 2：背 + 二头肌 🦈
✅ 坐姿划船：4组 x 12次（背部挺直，肩胛骨后缩）
✅ 高位下拉：4组 x 12次（想象要把手肘插进裤兜里）
✅ 哑铃弯举：4组 x 15次（不要借力晃身体哦 🚫）

3️⃣ Day 3：有氧或者主动休息 🧘‍♀️
✅ 椭圆机30分钟或者快走40分钟
✅ 微微出汗就好，不要太累，保持心情愉悦 💦

4️⃣ Day 4：腿部轰炸 🦵
✅ 徒手深蹲：4组 x 15次（膝盖尽量不要内扣）
✅ 腿举机倒蹬：4组 x 12次（重量适中，注意安全）
✅ 哑铃箭步蹲：3组 x 每侧10次（核心收紧 🧱）

5️⃣ Day 5：肩 + 腹 👱‍♀️
✅ 哑铃推举：4组 x 12次（直角肩练起来 💎）
✅ 哑铃侧平举：4组 x 15次（小重量找感觉）
✅ 平板支撑：4组 x 力竭为止（肚子酸酸的才有效 😂）

6️⃣ Day 6-7：好好休息，吃点好吃的 🍗
✅ 肌肉是在休息的时候生长的哦 🌙
✅ 睡个好觉，迎接下一周的训练 🛌

📍 组间休息控制在60-90秒 ⏱️ 别一边玩手机一边玩忘了时间 📱
📍 练完一定要拉伸！拉伸！拉伸！重要的事情说三遍 🧘‍♀️ 不然第二天真的会路都走不动，哭都没地方哭 😭
